# 📊 Analytical Reports & Deliverables Store

This folder holds all programmatically exported files, CSV reports, static PNG figures, and execution logs generated by the metrics engines.

## 📁 Key Assets
* **`fund_scorecard.csv`**: Ranks all 40 funds using percentile scorecard calculations across CAGR, Sharpe, Alpha, expense ratio, and drawdowns.
* **`alpha_beta.csv`**: Contains OLS regressed Alpha and Beta intercepts/slopes vs Nifty 100 daily returns.
* **`var_cvar_report.csv`**: Historical daily 95% Value at Risk (VaR) and Conditional VaR estimates.
* **`cohort_analysis.csv`**: Aggregated volumes and preferred categories for the 2024 vs 2025 cohorts.
* **`sip_continuity.csv`**: Flagged investors with average day-gaps exceeding 35 days (signaling churn risk).
* **`sector_hhi.csv`**: Sector concentration HHI indexes for equity holdings.
* **`worst_drawdown_ranges.csv`**: Peak, trough, and recovery metrics tracking worst historical drawdowns.
* **`rolling_sharpe_chart.png`**: Annualized rolling 90-day Sharpe ratio trend chart.
* **`sector_hhi_chart.png`**: Top equity fund HHI concentration bar plot.
* **`benchmark_comparison.png`**: Top 5 scorecard funds vs Nifty 50 and Nifty 100 normalized performance charts.
* **`metrics_computation.log`**: Step-by-step logs and exceptions written by the metrics calculation engine.
* **`email_delivery_mock.log`**: Local delivery verification file written by the offline mock smtp client.
