# 📺 Streamlit Dashboard Application Layer

This directory contains the presentation and frontend UI code for the Bluestock Mutual Fund Intelligence Platform.

## 📁 Directory Structure
* **[app.py](file:///D:/New%20folder/bluestock_mf_capstone/dashboard/app.py)**: The main entry-point script for the Streamlit dashboard. It implements the modern, high-fidelity dark-themed user interface, charts (Seaborn and interactive Plotly), quant scanners, time series ARIMA forecasts, stress test scenario simulators, and ML persona visualizations.

## ⚙️ Running the Dashboard
To launch the interactive dashboard locally, run the following command from the root workspace directory:
```bash
streamlit run dashboard/app.py
```
By default, the application will compile and open in your default browser at:
* Local URL: `http://localhost:8501`
